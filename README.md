
Proposal Title (tentative)
“State–Context Decoupling in Meditative Brains: A Multiscale EEG Analysis of Sensory Gating and Internal Attention”

Core Novel Idea (what’s actually new)
Instead of asking “How does meditation change EEG?”, we ask: How well does a meditative brain decouple internal state from external sensory context?
The novelty is to quantify sensory decoupling using context-invariant neural representations across visual conditions.
Meditators are hypothesized to:
●	Maintain a stable internal neural state
●	Suppress or gate irrelevant sensory input
●	Show reduced dependency on visual stimulation (grating, eyes open)
This lets us frame meditation as adaptive sensory gain control, not just relaxation.

Key Research Questions
1. Does meditation reduce neural sensitivity to visual context?
2. Are meditative EEG patterns invariant across sensory conditions?
3. Which neural scales (oscillatory, connectivity, complexity) best capture this invariance?

Methodological Innovation
We can do some mathematical modelling here. Instead of analysing conditions separately, treat them as domain shifts.
⮚	Context-Invariance Index (CII) <tentative> – NEW METRIC



Main Hypotheses
1. Context Invariance Hypothesis
●	Meditators show higher EEG representation similarity across visual conditions.
2. Sensory Gain Control Hypothesis
●	Visual gratings induce weaker occipital entrainment during meditation.
3. Top-Down Modulation Hypothesis
●	 Increased frontal→occipital directed connectivity during sensory stimulation.
